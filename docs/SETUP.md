# Setup Instructions

## Requirements
- Python 3.10+
- pip (Python package manager)
- Git

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shornellchabalala/mindmate-quest.git
   cd mindmate-quest
2. pip install -r requirements.txt
3. python src/app.py
4. http://127.0.0.1:5000
